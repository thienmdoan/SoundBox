# SoundBox
Website that lets you listen to new music.

Technologies:

Javascript

Node.js

HTML 5

CSS 3

RESTful API

Spotify API

Express

NPM

Nodemon

To start:

 1. npm install.
 2. npm run watch.
 3. Input your spotify access tokens in node.js as env variables.
 4. Server running on localhost:3000.
